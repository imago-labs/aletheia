# Compiled ABIs

> Auto-generated ABIs for `AletheiaPool.sol`. Do not edit by hand.

ABIs are produced by Hardhat compilation in the `contracts/` directory and committed here for downstream consumers (frontend, backend, integration partners).

## Regenerate

```bash
cd ../contracts
npm install
npx hardhat compile
cp artifacts/contracts/AletheiaPool.sol/AletheiaPool.json ../abis/
```

## Versioning

Each ABI is suffixed with the contract version it was compiled from (for example `AletheiaPool.v2.json`). When the ABI changes incompatibly, the version is incremented.

ABIs will be published with the first alpha release.
